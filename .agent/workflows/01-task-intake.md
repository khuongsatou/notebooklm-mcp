# Workflow: Task Intake

## Mục tiêu

Nhận task mới và đưa về trạng thái sẵn sàng xử lý.

## Các bước

1. PM ghi task vào `.manager/current_task.md`.
2. Xác định mục tiêu, phạm vi, deadline, priority và owner.
3. Chọn skill cần dùng trong `.agent/skills`.
4. Xác định rule bắt buộc trong `.agent/rules`.
5. Ghi dòng mới vào `.manager/iteration_log.md`.

## Done khi

Task có owner, scope và next action rõ ràng.
