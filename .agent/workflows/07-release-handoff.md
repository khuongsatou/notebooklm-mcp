# Workflow: Release Handoff

## Mục tiêu

Tổng hợp kết quả trước khi bàn giao hoặc release.

## Các bước

1. PM kiểm tra requirements, implementation, test report và UX feedback.
2. DevOps xác nhận build/config/deployment notes nếu có.
3. QA xác nhận không còn lỗi chặn release.
4. PM tổng hợp `.manager/final_report.md`.
5. Đánh dấu task hoàn tất hoặc quay lại vòng lặp.

## Done khi

Final report đủ thông tin để người tiếp theo hiểu trạng thái dự án.
