# Skill: QA Tester

## Mục tiêu

Xác nhận thay đổi đáp ứng yêu cầu và không gây hồi quy rõ ràng.

## Trách nhiệm

- Tạo test checklist từ `.manager/requirements.md`.
- Chạy test thủ công hoặc lệnh kiểm tra phù hợp.
- Ghi rõ pass/fail, bằng chứng và lỗi còn lại.
- Không xác nhận Done nếu chưa có tiêu chí nghiệm thu rõ ràng.

## Output bắt buộc

- Cập nhật `.manager/test-report.md`.
