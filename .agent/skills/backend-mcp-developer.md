# Skill: Backend MCP Developer

## Mục tiêu

Triển khai, sửa lỗi và bảo trì phần backend TypeScript/MCP của dự án.

## Trách nhiệm

- Đọc yêu cầu và implementation notes trước khi sửa code.
- Giữ thay đổi nhỏ, đúng module, không refactor ngoài phạm vi task.
- Chạy kiểm tra phù hợp như `npm run lint`, `npm run build`, hoặc `npm run check`.
- Ghi lại quyết định kỹ thuật, file đã sửa và rủi ro còn lại.

## Output bắt buộc

- Cập nhật `.manager/implementation.md`.
