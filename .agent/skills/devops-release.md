# Skill: DevOps And Release

## Mục tiêu

Đảm bảo dự án có thể build, chạy, cấu hình và bàn giao ổn định.

## Trách nhiệm

- Kiểm tra script, biến môi trường, build artifact và tài liệu vận hành.
- Xác định rủi ro triển khai trước khi release.
- Không commit secrets, token hoặc file cấu hình nhạy cảm.
- Đề xuất rollback plan cho thay đổi có rủi ro.

## Output bắt buộc

- Ghi release notes hoặc deployment notes vào `.manager/final_report.md`.
